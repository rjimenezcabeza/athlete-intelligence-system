# Análisis Competitivo AIS

## Competencia principal: RP Hypertrophy App
URL: https://rpstrength.com/pages/hypertrophy-app
Precio: ~$14.99/mes
Fortalezas: Brand Dr. Mike Israetel, metodología científica, app nativa
Debilidades: UI confusa/sobrecargada, sin IA conversacional, solo metodología RP, sin importador

## Jefit Elite
Precio: ~$7.99/mes
Fortalezas: UX simple, precio bajo, comunidad
Debilidades: Sin IA, sin metodología científica, básico en progresión

## Posicionamiento AIS
AIS ocupa el cuadrante: Metodología científica + UX simple + IA conversacional

Ventajas competitivas:
1. IA Coach conversacional — ningún competidor lo tiene
2. Multi-metodología — no atado a una sola escuela
3. Importador inteligente — onboarding sin fricción
4. Wearables en el loop de progresión
5. Precio paridad con RP, UX superior
6. Mobile-first extremo: set en menos de 5 segundos
7. Coach + Athlete mode desde el diseño

Conclusión: AIS no compite siendo otra app de tracking. Compite siendo el sistema operativo del atleta.
