# Roadmap AIS — Fases 1-7

## Fase 1 — MVP (Sprints A-D) — EN CURSO
Auth, DB 12 tablas, Motor progresión, IA Coach, Wearables, Gráficas, Push
KPI: 100 usuarios Pro

## Fase 2 — Importador Inteligente
Upload imagen/PDF/Excel/Word/texto → datos estructurados automáticamente via IA.
KPI: Onboarding desde cero en menos de 5 minutos.

## Fase 3 — Coach Mode (B2B)
Dashboard multi-atleta para coaches. Billing por seat de atleta. Roles: admin/coach/atleta.
KPI: Primer coach B2B con 5 o más atletas activos. €5k MRR.

## Fase 4 — Nutrición Integration
Tracking macros y calorías. Sync MyFitnessPal/Cronometer. Fases bulk/cut/recomp.

## Fase 5 — Marketplace de Planes
Coaches publican planes (free o de pago). Revenue split 70% coach / 30% AIS.

## Fase 6 — Analytics Avanzados
ML para predicción 1RM. Análisis patrones fatiga. Export datos científicos.
KPI: 1.000 usuarios Pro. €50k MRR.

## Fase 7 — Plataforma
API pública, white-label para gyms, CV análisis técnica, AIS Enterprise.
